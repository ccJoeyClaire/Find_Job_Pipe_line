职位简介

Job Description

A Configuration Specialist you will be responsible for the following:

  • Perform scoping workshops and production of associated scope documentation.
  • Assist in customer server set up and configuration utilising the appropriate Cimteq template installation database.
  • Produce training material and perform training workshops.
  • Provide technical input to customer ERP Integration discussions in conjunction with utilising the Cimteq Standard API Integration solution.
  • Assisting customers to implement the Cimteq product suite.
  • Completion of Implementation activities in line with the agreed timescales, deadlines, and business KPI’s.
  • Adhering to best practices used within the Implementations team.

Responsibilities

Day to day you will be a working member of the Implementation team which includes, scoping, training, and supporting customer implementations throughout the implementation period.

You will have customer facing experience, interacting with customers as required.
  • Subject matter expert in product suite functionality.
  • Assist the Sales Team by providing customer demonstrations of product functionality.
  • Assist in pre-sales scoping, estimation, and quotation activities in relation to customised development solutions.
  • Aid the PMO with project planning and scope planning.
  • Analysing requirements and produce technical and business documentation.
  • Delivery of workshops, training and implementation activities to timescales, and deadlines.
  • Providing excellent customer service, interacting with customers as required.
  • Adhering to best practices used within the implementation team.
  • Perform any other tasks or duties assigned by your manager or the Leadership team.
  • Actively commit to self-development.
  • To attend all internal and external meetings and training courses.
  • To share skills and knowledge within the team.

Qualifications
  • Proficient in Japanese (Must)
  • Knowledge of software and application design and architecture.
  • Knowledge of relational database principles and SQL language.
  • Knowledge of Web based programming technologies, e.g. HTML, JavaScript, jQuery, XML.
  • Knowledge of the cable and wire industry. (Optional)
  • Experience for developing/using ERP, MES… (Optional)

The role will involve use of the following software:
  • Eclipse IDE
  • Microsoft SQL Server
  • Java
  • JavaScript
  • Oracle
  • MySQL
  • Apache Tomcat
  • Jenkins
  • Crystal Reports
  • Jira
  • Microsoft Office
  • XSLT
  • XSLT-FO
  • Microsoft Windows & Unix O/S
  • XML & JSON
