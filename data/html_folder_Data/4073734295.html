职位简介

Schaeffler is a dynamic global technology company and its success has been a result of its entrepreneurial spirit and long history of private ownership. Does that sound interesting to you? As a partner to all of the major automobile manufacturers, as well as key players in the aerospace and industrial sectors, we offer you many development opportunities.

Your Key Responsibilities

Hybrid Module Testing Engineer (PEU)

  • Implement DHT Transmission / Hybrid Module PEU test according to project request.
  • Set-up standard processes for the testing activities
  • Decompose testing requirements from customers, provide clear inputs to the test supplier.
  • Communicating with test supplier to help them finish the test preparation and controlling test quality, maintain test status.
  • Strongly trouble solve skill, give and seek support proactively.
  • Test data process, test sample analysis and write test reports and presentation about the testing results.
  • Take responsible to track PEU DV&PV test status.
    • Organize or join regular meetings with PEU supplier’s testing engineer and test supplier to review DV test status.
    • Onsite check regularly, evaluate test status in test supplier with PEU supplier’s test engineer, escalate issue and feedback status to team.
    • Assurance of correct return part analyze.
  • Evaluate PEU test quality.
    • Evaluate if test spec meets test requirement.
    • Evaluate if test process meets test spec.
    • Evaluate if test report meets test spec.
  • On-site support for PEU test trouble shooting.
  • Summarize test lessons learn and feedback to team.
  • Other on-site urgent items checking.
  • Support internal PEU testing.
  • Other tasks related to PEU DVP
Your Qualifications

Bachelors’ Degree or above

Fluent in English, fluent in German will be preferred.

At least 3 year working experience in PEU testing,

Good teamwork with strong communication skills

Strong handling of Microsoft office

Familiar with ISO16750 GBT18488 VW80000 standards.

Proficient in CANoe/CANanalyzer/INCA and other software testing tools.
