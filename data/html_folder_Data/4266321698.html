职位简介

Responsibilities:

1. Lead the development of a Real-Time Operating System (RTOS) for humanoid robots, achieving microsecond-level motion control loops (e.g., whole-body dynamic balancing control).

2. Design a multi-sensor fusion framework (vision, LiDAR, IMU, force sensing), optimizing data synchronization accuracy to within 0.1ms.

3. Develop a distributed computing architecture supporting parallel control and hot-swapping for 100+ joints.

4. Build a functional safety system (Functional Safety Level ≥ SIL2), implementing hardware failure self-recovery mechanisms.


Qualifications:

1. Master's degree or higher in Computer Science/Robotics, with 8+ years of experience in embedded systems development.

2. Proficient in C++17/ROS2 framework, with expertise in real-time Linux kernel customization (e.g., PREEMPT_RT optimization).

3. Experience in developing motion control systems for humanoid robots (e.g., engineering implementation of ZMP/MPC algorithms).

4. Familiarity with EtherCAT/TSN network protocols and experience implementing systems with control frequencies of 1kHz or higher.

5. Led the core control system development for at least 2 robot products (must provide demo videos).


人形机器人软件专家

岗位职责：

2. 精通 C++17/ROS2 框架，掌握实时 Linux 内核定制（如 PREEMPT_RT 优化）。

4. 熟悉 EtherCAT/TSN 网络协议，实现过 1kHz 以上控制频率系统。
