Responsibilities

Role Overview

We are looking for a skilled Senior Full Stack Software Engineer who excels in both backend and frontend development, with deep expertise in Java and frontend technologies as well as the knowledge of relational database, such as MySQL to drive the delivery of high-quality software solutions. This role requires fluency in English (written and spoken) to collaborate with global teams, good problem-solving ability, familiar with Agile methodologies and a self-driven mindsets to lead projects from concept to deployment. You will be a key player in cross-functional squads, fostering teamwork and innovation.

What You’ll Do

You will work across the entire stake with a strong emphasis on both fronted features and backed development using Java, you may also contribute your insight regarding how to design the database. Besides the development, you’ll also need to investigate the issues and resolve critical issue may be in a short period of time.

Key Responsibilities

  • Full Stake Development
  • Technical design, develop and maintain robust backend services and APIs using Java like Spring Boot.
  • Develop frontend applications with frameworks like Angular or Vue.js (TypeScript/JavaScript).
  • Ensure seamless integration between frontend and backend systems.
  • Ensure end-to-end system integration, performance and security.
  • Collaboration
  • Participate in agile ceremonies (sprint planning, daily standups, retrospectives) to deliver iterative value.
  • Collaborate with product owners, designers and QA engineers to refine user stories and acceptance criteria.

Qualifications
  • Requirements:
    • Education: Bachelor’s / Master’s degree in computer science, engineering or equivalent practical experience.
    • Experience: 5+ years in full stack development, including 2+ years focused on Java backend systems.
    • Ability to communicate effectively with cross-functional/cross-geography teams
    • Fluent in English (both technical documentation and verbal communication).
    • Self-motivated, results-oriented, and adaptable to shifting priorities.

  • Technical Skills:
    • Frontend: Proficiency in HTML5, CSS3, JavaScript/TypeScript; and modern frameworks (Angular or Vue.js), familiar with NodeJS and the build tools such as webpack or grunt.
    • Backend: Mastery of Java 11+, Spring ecosystem (Boot, Security), JavaEE, SQL databases (MySQL) and microservices.
  • Collaboration:
    • Proven track record in Agile/Scrum environments with tools like Jira, Confluence.
    • Strong team player with the ability to motivate peers and resolve conflicts constructively.
  • Preferred:
    • Certifications: Java SE Professional or Scrum Master (CSM).
    • Hands on experience with AWS/Azure, Docker, Kubernetes, CI/CD pipelines (Jenkins, Github CI).
    • Familiarity with monitoring tools and frontend testing framework.
