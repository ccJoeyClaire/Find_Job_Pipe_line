职位简介

此职位来源自其他求职网站。

该职位来源于猎聘 【岗位职责】 1. 负责公司数据库系统的架构设计、部署、维护与优化，确保高可用性与高性能； 2. 管理和维护多种数据库系统，包括关系型数据库（如 MySQL、PostgreSQL、Oracle）和非关系型数据库（如 MongoDB、Redis、Cassandra）； 3. 参与数据库容量规划、性能调优、故障排查与恢复策略制定； 4. 设计并实施数据备份、恢复、容灾方案，保障数据安全与业务连续性； 5. 支持开发团队进行数据库设计与查询优化，推动数据库规范化与自动化运维； 6. 跟踪数据库技术发展趋势，持续提升平台能力与运维效率。 【任职要求】 1. 计算机相关专业本科及以上学历，5年以上数据库管理经验； 2. 精通至少一种主流关系型数据库（如 MySQL、PostgreSQL、Oracle、SQLServer），具备数据库性能调优与故障处理经验； 3. 熟悉至少一种主流非关系型数据库（如 MongoDB、TiDB、StarRocks/Doris、ElasticSearch），了解其应用场景与运维要点； 4. 熟悉数据库高可用架构、分布式部署、数据同步与迁移技术； 5. 熟悉 Linux 操作系统及 Shell 脚本，具备自动化运维能力； 6. 具备良好的沟通能力与团队协作精神，能支持多业务线数据库需求。 7. 能独立搭建分布式数据库（包括关系型和非关系型） 【加分项】

  • 有云数据库（如 AWS RDS、Azure SQL、Alibaba RDS）运维经验者优先；
  • 熟悉 DevOps、数据库监控工具（如 Prometheus、Grafana）者优先；
  • 有数据库安全、审计、加密等方面经验者优先。
