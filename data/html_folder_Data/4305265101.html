Summary

The Engineer, Software develops, debugs, tests, deploys and supports code to be deployed in systems/products/equipment for various applications. They write, debug, maintain, and test software in various common languages and for software at various levels in the hierarchy (from Firmware to Application). Software creation follows an agreed to development process (such as Agile, Scrum, etc.) and complies with the product life cycle development (phase/gate deliverables).

The Engineer, Software works in cross functional teams with other designers, customers, manufacturing engineering and project leadership to ensure robust and high quality product development. Enhance designs with feedback from reviews in areas such as manufacturing, test, supply chain, reliability, industrial design and simulations.

Detailed Description

Main Responsibility

  • Working as Software Test Development Engineer for Server or Comms product design validation
  • Perform Test Automation and setting up frameworks on multiple application platforms like Server and networking.
  • Responsible for automation test development for FW and system level testing.
  • Support designer bug analysis/duplication, issue tracking and reporting.
  • Duplicate and solve Customer feedback issues.
  • Ensure the quality of design products and minimize defects escapes to customers
  • Ensure the validation task be finished timely by well scheduled time frame, improving the test efficiency and automating the test execution.
  • Must work closely and constantly communicate with various international teams, including development, QA, documentation, release management, marketing, tech support and customers
  • Develop and compile test utility, application used under Linux/Unix environments
  • Test equipment and tool purchase forecast for new technology pursuit.

Knowledge/Skills/Competencies

Person Background
  • Bachelor’s degree in CS or similar.
  • Working experience: 5 + years.
  • 8+ years solid experience in software / test development.
  • Familiar with Python/Shell script development
  • Knowledge of hardware diagnostics or test techniques
  • Familiar with Windows server 2008/2012/2016, Suse 32/64, Redhat, Fedora, Ubuntu etc
  • Knowledge of IPMI ,ISCSI and server/storage/comms management, such as the concept of KCS, IPMB, I2C, RGMII, KVM, SOL.
  • Knowledge of one or more of the following standards/technologies: PCI, ACPI, IPMI, SMBIOS, SATA, SSD, SMART, PCI Express.
  • Knowledge of software testing and test case development
  • Knowledge of Linux Kernel or device driver environment.
  • Good oral and writing skills in English
  • Skills in compile test utility and an application is a plus.
