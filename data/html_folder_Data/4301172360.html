关于我们

Bybit 成立于 2018 年 3 月，是全球增长最快的加密货币衍生品交易所之一，现已拥有超过 7000 万注册用户。我们为加密货币交易者提供专业平台，配备超高速撮合引擎、优质客户服务以及多语言社区支持。业务涵盖创新型的现货与衍生品交易、挖矿与质押产品，以及 API 接口服务，面向全球零售与机构客户，致力于成为新兴数字资产领域最值得信赖的交易所。


我们的核心价值观塑造了我们。我们倾听用户、关心用户、不断优化，力求为用户打造更快、更公平、更人性化的交易环境。我们的创新型、高度先进、用户友好的平台基于顶级基础设施自研打造，旨在为用户提供业内最安全、最快速、最公平、最透明的交易体验。秉承以客户为中心的理念，我们提供 7x24 小时、多语言专业客服，及时高效地为用户提供支持。


如今，Bybit 已成为业内最值得信赖、最可靠、最透明的加密货币衍生品交易平台之一。


岗位职责

1. 负责集团级公共数据仓库的建设与优化（离线&实时），沉淀统一的数据资产

2. 主导标签体系建设，打造实时和离线标签能力，支持业务活动与精细化触达

3. 设计与开发用户&产品特征库，协同算法团队推动个性化推荐与智能应用落地

4. 深入理解业务需求，完成数据仓库建模及数据资产化设计，保障平台的数据可复用性与扩展性

5. 建设数据清洗、质量监控与告警体系，持续提升数据交付的准确性与稳定性


任职要求

1. 本科及以上学历，计算机、数学、统计或相关专业

2. 2年以上数据仓库开发经验，精通 SQL，具备丰富的 SQL 优化经验，熟悉 SparkSQL 等大数据计算引擎

3. 深入理解数仓与BI 体系，包括 ETL、数据仓库架构、OLAP、多维建模等核心知识

4. 至少掌握 Python、Scala、Java 中的一门编程语言，具备良好的工程化能力

5. 有 Hadoop、Hive、Spark、Flink、Kafka、Trino、Doris、TiDB 等大数据生态经验者优先

6. 工作严谨，具备优秀的逻辑思维与学习能力，善于沟通与团队协作，能够独立推动复杂问题的解决
