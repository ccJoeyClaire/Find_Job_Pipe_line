职位简介

About Us

We’re a fast-moving, AI-leveraged rapid prototyping lab creating innovative products in martech, fintech, web apps, websites, crypto, and more. Our developers are the backbone of our mission to build and iterate at speed, delivering solutions that blend cutting-edge tech with real-world impact.


Job Summary

As a Software Developer, you’ll build and iterate on prototypes and MVPs across our diverse product stack. Working closely with product managers and marketing professionals, you’ll leverage AI tools and frameworks to deliver scalable, secure, and high-performance solutions in a rapid development environment.


Key Responsibilities
  • Utilize AI-driven development tools like Cursor, GitHub Copilot, Replit AI, or Tabnine to streamline coding, debugging, and iteration on prototypes.
  • Develop and maintain web apps, websites, and backend systems using modern frameworks (e.g., React, Node.js, Python, etc.).
  • Integrate AI/ML models into products (e.g., recommendation systems for martech, fraud detection for fintech).
  • Build and test smart contracts or blockchain-based features for crypto-related projects.
  • Optimize code for performance, scalability, and security in cloud environments (e.g., AWS, Azure).
  • Contribute to architecture decisions and adopt best practices in an agile, iterative workflow.


Qualifications
  • Familiarity with AI-driven development tools (e.g., code generation, automated testing) is a must.
  • 2+ years of software development experience, with proficiency in at least one modern stack (e.g., JavaScript, Python, Go, Rust). Fresh Graduates with relevant experience leveraging AI tooling will be considered.
  • Experience with REST APIs, databases (SQL/NoSQL), and cloud platforms.
  • Familiarity with AI/ML integration (e.g., TensorFlow, PyTorch, or API-based AI services).
  • Knowledge of blockchain development (e.g., Solidity, Rust, Move) is a plus but not a must.
  • Strong problem-solving skills and ability to thrive in a fast-paced, prototype-driven setting.
  • Bonus: Experience in martech (e.g., adtech APIs), fintech (e.g., payment gateways), or crypto.
