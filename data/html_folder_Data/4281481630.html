AI工程师&RPA工程师

BizLink Group · 中国 广东省 深圳 (现场办公)

职位简介

职位描述：

作为一名专注于计算机视觉的人工智能和大数据工程师，您将领导人工智能项目的开发和实施，从项目启动到部署。这包括理解项目需求、选择合适的解决方案、收集和标记数据、训练、测试以及部署用于生产的模型。项目主要涉及智能制造中的图像处理、图像分类和目标检测，用于缺陷检测。此外，您的兼职职责还包括使用 UiPath RPA 机器人自动执行重复性任务、参与 RPA 流程设计以及维护 RPA 流程。您还将负责将应用程序封装到 API、桌面应用程序或 Web 应用程序中。

岗位职责：

A、人工智能 - 计算机视觉

1. 执行以图像处理、分类和对象检测为重点的人工智能项目；

2. 收集、标记和预处理图像数据以训练 AI 模型；

3.使用深度学习框架训练、测试和部署计算机视觉模型；

4.优化 AI 模型的性能和效率；

5.将应用程序开发和部署为 API、桌面或 Web 应用程序.

B、RPA （兼）

1.使用 UiPath RPA 机器人自动执行重复性任务；

2.参与RPA流程设计及可行性分析；

3.开发、调整和优化 RPA 工作流程；

4.维护RPA流程，排除故障，确保稳定运行。

岗位要求：

1、2年相关工作经验；

2、能够使用英语和普通话进行交流；

3、具有图像处理、图像分类、对象检测/跟踪、图像分割、光学字符识别 (OCR) 以及 ML/DL

4、.模型训练和部署方面的经验。

5、了解AI模型优化。

6、熟悉 Docker 等容器化技术。

7、熟练掌握至少一种深度学习框架：TensorFlow-Keras、PyTorch。

8、熟悉 Pandas、NumPy、Scikit-learn、Matplotlib、Flask、Django、OpenCV 等库。

9、具有 Tkinter、PySimpleGUI 或 Streamlit 等 GUI 开发库的经验。

10、优先具有开发 API 或在服务器上部署模型的经验。

9.优先考虑具有数据分析、ETL 流程和 CI/CD 经验的人。

11、熟悉Oracle、MongoDB等数据库。

12、优先熟悉自然语言处理 (NLP) 和生成式人工智能 (Gen AI)。

13、较强的自学能力、解决问题能力和良好的团队合作能力。
