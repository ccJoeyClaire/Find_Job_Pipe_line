Work you'll do

岗位职责：
  • 数据开发与建模：
    • 负责使用 Python 进行数据采集、清洗、分析和建模，支持业务决策。
    • 集成 LangChain 和 LangGraph 框架，构建高效的文本处理和知识图谱应用。
  • AI 大模型开发与优化：
    • 参与生成式 AI（GAI）大模型的开发、调优和部署。
    • 使用 LangChain 与 GAI 模型集成，优化模型推理速度和生成质量。
  • 框架与工具集成：
    • 负责将 LangChain 等工具集成到公司现有数据平台和 AI 系统中。
    • 探索和引入新技术，提升数据开发效率和系统性能。
  • 数据_pipeline 开发：
    • 构建和优化数据 pipeline，从数据源到模型输入的全流程处理。
    • 使用 Python 和相关库（如 Pandas、NumPy、Dask）处理大规模数据集。
  • 跨团队协作与技术支持：
    • 与数据科学家、业务分析师和产品经理协作，理解需求并转化为技术解决方案。
    • 提供技术支持，确保数据产品的稳定运行和性能优化。
We are looking for someone with
  • 任职要求：
      • 技术能力：
        • 熟练掌握 Python，有丰富的数据开发经验（至少 3 年）。
        • 熟悉 LangChain、LangGraph 或类似框架，有相关项目经验。
        • 了解生成式 AI（如 GPT）的工作原理，有 GAI 模型开发或调优经验者优先。
      • 数据处理与分析：
        • 熟悉常见的数据处理库（Pandas、SQL）和机器学习框架（如 PyTorch、Hugging Face）。
        • 了解数据建模、知识图谱构建或图数据库（如 Neo4j）经验者优先。
      • 系统集成与开发：
        • 熟悉 CI/CD 流程，有 Docker、Kubernetes 等容器化经验者优先。
        • 了解大数据处理平台（如 Hadoop、Spark）或云平台（如 AWS、GCP）者优先。
      • 沟通与协作能力：
        • 能够与跨职能团队有效沟通，将业务需求转化为技术方案。
        • 有良好的逻辑思维能力和问题解决能力。
