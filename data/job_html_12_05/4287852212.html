职位简介

Job Description

  • Responsible for accelerating and optimizing deep learning models on automotive platforms, designing GPU-based high-performance algorithm libraries to support downstream algorithm R&D.
  • Implement core algorithms for autonomous driving systems, collaborate with algorithm researchers on algorithm implementation and deployment, and optimize algorithm efficiency at the code and instruction set levels.

Requirements
  • Proficient in CUDA principles, capable of independently performing code hotspot profiling and developing optimization strategies.
  • Strong programming skills in Python and C++, with solid knowledge of common algorithms and data structures.
  • Excellent engineering capabilities, including robust code architecture design skills, and proficiency with tools like Git, SSH, and CMake.

Preferred Qualifications
  • Award-winning experience in computer/informatics competitions (e.g., ACM/ICPC, CCPC, NOI, IOI).

小米公司正式成立于2010年4月，是一家专注于高端智能手机、互联网电视以及智能家居生态链建设的创新型科技企业。“让每个人都能享受科技的乐趣”是小米公司的愿景。小米公司应用了互联网开发模式开发产品的模式，用极客精神做产品，用互联网模式干掉中间环节，致力于让全球每个人，都能享用来自中国的优质科技产品。
2017年，小米跨越千亿营收的门槛，小米手机销量成功逆转，始终坚持技术创新，发布了澎湃S1处理器芯片，小米MIX 2继续引领全面屏技术; 通过庞大而先进的智能硬件生态链布局和持续投入的云服务，建起了全球最大的IOT平台，并且在国内AI领域率先取得落地产品的成功; 国际化迅猛发展，在进入的70多个国家和地区中16个国家销量进入了前5位，其中印度连续两个季度稳居市场份额第一。
