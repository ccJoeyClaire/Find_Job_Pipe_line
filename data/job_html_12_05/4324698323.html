Data Scientist

Robert Half · 中国 上海市 上海市 (现场办公)

职位简介

Job Responsibilities

  • Recommendation System Optimization: Responsible for the development and optimization of personalized recommendation algorithms for e-commerce platforms, including product recommendations, content recommendations, event recommendations, and service recommendations, aiming to improve user conversion rates and retention.
  • Search Ranking Algorithms: Design and implement e-commerce search ranking algorithms to optimize search relevance, click-through rates, and conversion rates, enhancing the user search experience.
  • Sales Forecasting and Inventory Optimization: Develop time-series algorithms such as sales forecasting to support supply chain optimization, store scheduling, intelligent replenishment, and other business decisions.
  • Large Model Applications: Familiar with the use cases and capability boundaries of state-of-the-art large models (language/multimodal), with skills in engineering calls, prompt engineering, context engineering, and basic fine-tuning.
  • Algorithm Engineering: Proficient in building data pipelines for algorithm models, code deployment, and algorithm serving, with data engineering and backend engineering capabilities to support algorithm implementation.
  • Open Source Integration: Capable of applying and customizing cutting-edge open source projects, skilled at identifying synergies between outstanding open source technologies and business needs.

Job Requirements

1. Education: Bachelor's degree or above in Computer Science, Artificial Intelligence, Mathematics, Statistics, or related fields
2. Work Experience: More than 2 years of algorithm development experience; candidates with backgrounds in e-commerce or the internet industry are preferred
3. Technical Skills:
  • Proficient in at least one programming language such as Python, Java, or C++
  • Understanding of current SOTA large model trends and familiar with large model application platforms (such as Bailian, Qianfan, etc.)
  • Familiar with mainstream machine learning frameworks (TensorFlow, PyTorch, Scikit-learn, etc.)
  • Solid foundation in machine learning and deep learning theories, familiar with commonly used algorithms (LR, GBDT, DNN, Graph Neural Networks, etc.)
  • Familiar with big data processing technologies (Hadoop, Spark, Flink, etc.)
  • Good foundation in data structures and algorithms
