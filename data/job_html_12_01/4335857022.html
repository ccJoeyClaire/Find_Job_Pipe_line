职位简介

岗位职责

  • 参与设计并实现基于大模型的AI应用方案；
  • 参与构建与优化 RAG（Retrieval-Augmented Generation）系统，协助数据检索、模型调优与性能优化；
  • 学习并掌握 Transformer 架构及主流预训练模型（如BERT、Llama、GPT等），探索其在不同业务场景中的应用；
  • 负责数据集的收集、清洗与标注工作，支持模型训练与测试实验，并参与基于实验反馈的模型改进；
  • 与跨职能团队（产品、算法、工程）协作，支持AI模型的API部署与产品功能落地；
  • 持续关注 Web3、区块链与AI结合方向 的最新研究进展，参与创新应用探索项目。

任职要求
  • 计算机科学、人工智能、机器学习、数据科学或相关专业，本科及以上学历；
  • 具备扎实的编程基础，熟悉 Python / Java / Go 等至少一种语言；
  • 了解 Transformer 模型原理，对LLM、大语言模型或RAG系统有基础理解；
  • 熟悉数据处理与分析流程，如数据预处理、ETL、SQL、Pandas、向量数据库等；
  • 了解深度学习框架（PyTorch / TensorFlow）；
  • 具备良好的学习能力、逻辑思维与问题解决能力，能主动探索AI与业务结合的创新应用；
  • 有以下经历者优先：
  • 曾参与RAG或大模型微调项目（课程、科研、实习均可）；
  • 了解Web3 / 区块链基础知识；
  • 在AI竞赛、开源项目中有贡献记录（如GitHub、Kaggle）
